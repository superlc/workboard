## Capability: stats-view

Dedicated view at `/tasks/stats` for viewing work analytics.

## Requirements

### STAT-1: Full-Width Stats
- Render the `TaskStats` component at full content width.
- Takes advantage of the larger space to display charts more clearly.

### STAT-2: No Input or Calendar
- This view must NOT render the task input or calendar components.
- Focused solely on analytical/statistical content.
